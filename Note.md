# HuyCommerce-note
## 1. Chức năng của từng file:
* `register.html` & `login.html`: Hai trang để tạo tài khoản rồi đăng nhập vào web.
* `home.html`: Trang chính hiện mấy món đồ mình bán. Nhớ là phải đăng nhập rồi mới vào đây được, không là bị đuổi ra ngoài trang login đấy.
* `style.css`: Chỗ mình chỉnh màu mè, font chữ cho đẹp.
* `main.js`: Chỗ xử lý code cho việc đăng ký với đăng nhập. Dữ liệu user mình đang lưu tạm ở localStorage trên trình duyệt luôn.
* `home.js`: File này giờ bỏ trống rồi, vì mình chuyển hết đống sản phẩm sang viết trực tiếp bằng Bootstrap cho nó nhanh.
## 2. Web làm được những gì?
* **Đăng ký / Đăng nhập:** Tạo tài khoản xong thì dùng nó để vào xem hàng. 
* **Bảo vệ trang chủ:** Ai chưa đăng nhập mà đòi vào xem hàng là web tự chuyển hướng về trang đăng nhập ngay.
* **Xem sản phẩm:** Dùng Bootstrap làm mấy cái khung (Card) nhìn cho nó "xịn" và chuyên nghiệp. Nó tự co giãn đẹp trên cả điện thoại lẫn máy tính.
* **Đăng xuất:** Thoát ra cái là sạch sẽ, muốn vào lại phải login lại từ đầu.
